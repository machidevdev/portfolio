# Shishi

## About

Mint dApp for ShiShi, a collection of 3333 unique NFTs made by the artist [@shiro](https://x.com/shiro57102).

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Rainbowkit and Wagmi
- Github Actions
